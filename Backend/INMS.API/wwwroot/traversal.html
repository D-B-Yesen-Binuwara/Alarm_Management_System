Traversal (DFS / BFS)

Traversal algorithms are used to explore connected devices.

BFS (Breadth First Search)

  • Uses Queue
  • Visits devices level by level
  • Used for Impact Analysis

DFS (Depth First Search)

  • Uses Stack or Recursion
  • Explores deeply before backtracking

Why BFS is Used?

BFS ensures all directly and indirectly connected devices are identified.
