# Alarm_Management_System
web-based system designed to model and simulate fault detection, alarm handling, and impact analysis within a telecom service provider’s network. The system represents a hierarchical telecom topology and enables realistic simulation of power-related alarms, node failures, and downstream network impact.
The project focuses on demonstrating how modern telecom Network Operations Centers (NOC) monitor distributed infrastructure, identify faults, and analyze service disruptions using centralized systems. While the system does not interact with real telecom hardware, it logically simulates real-world operational behavior, communication flows, and monitoring techniques.
